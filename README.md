## About

The FeedReader is a web-based application that reads RSS feeds.
In this project we had to add tests checking the functionality of the code using Jasmine(v2.1.2). 

## How to run it

1) Download or Clone the repository
2) Unzip the file if you have downloaded a ZIP
3) Open the folder and locate index.html
4) Open index.html in your browser 

## Dependencies

*   Jasmine v2.1.2
*   jQuery v2.1.1
*   Handlebars.js v2.0.0
*   Google Feed API
